# LearnUpon with AngularJS

> Test repo to connect to the LearnUpon API using AngularJS.

___

## Usage

### Install node modules
```bash
  npm install
```

### Build once
```bash
  gulp
```

or

```bash
  gulp build
```

### Build on file changes
```bash
  gulp build-watch
```

### Clean/Remove build folder
```bash
  gulp clean
```

# LearnUpon with AngularJS

> Test repo to connect to the LearnUpon API using AngularJS.

___

## Usage

### Install Node Module Dependencies
```bash
  npm install
```

### Run Webserver
```bash
  npm start
```

### Clean & Build (Recommended)
```bash
  gulp
```

### Build

```bash
  gulp build
```

### Build when Files Changed
```bash
  gulp build-watch
```

### Clean Build Folder
```bash
  gulp clean
```
